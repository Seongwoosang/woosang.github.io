사용된 웹 기술

  1. HTML

    장점 :
    무료로 사용이 가능하며 문법이 쉽다.
    마크업 언어로 클라이언트-서버 간의 빠른 정보 전달이 가능하다.
    시스템과 운영체제가 독립적이다.

  2. CSS

    장점 :
    HTML의 기능을 확장하며 문법이 쉽다.
    양식을 모듈화해 전체를 같은 문서양식으로 구성할 수 있다.
    사용환경에 영향을 거의 받지 않는다.

  3. JAVASCRIPT

    장점 :
    컴파일 과정이 따로 필요가 없으며 바로 화면에 적용이 가능합니다.
    인터프리터 언어로서 동적이며 타입을 명시할 필요가 없습니다.
    객체 기반의 스크립트 언어입니다.
